MARKQUERY SKELETON

Spider's Web v1.0.0 for docpad

Customize your contents. Markquery is markup library for web contents.

  • Homepage http://markquery.github.io
  • Korean page: http://markquery.kr
  • Twitter: @markquery

Getting Started

  1. Install DocPad

  2. Clone the project and run the server

    git clone https://github.com/markquery/docpad-spidersweb.git
cd docpad-spidersweb
npm install
docpad run

  3. Open http://localhost:9778/

  4. Start hacking away by modifying the src directory
